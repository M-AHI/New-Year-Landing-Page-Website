## 2. Project Overview

This project is a **static New Year themed landing page website**.
It is designed to practice **basic web development skills**, especially HTML structure, CSS styling, layout design, and use of images.

The website focuses on:

* Attractive visual design
* Banner sections
* Icons and images
* Clean and simple layout

This project is suitable for **beginners** who are learning frontend web development.

---

## 3. Problem Description

### What problem does this project solve?

Beginners in web development often struggle to:

* Create a complete webpage layout
* Use CSS for proper styling
* Organize assets like images and stylesheets
* Build visually appealing static websites

This project helps solve those problems by giving hands-on experience in building a **real-looking website** from scratch.

---

### Why is this project important in real life?

* Websites are essential for businesses, events, and promotions
* Landing pages are widely used for marketing campaigns
* Learning static website design is the **first step** toward full-stack development

This project builds a strong foundation for future projects like:

* Portfolio websites
* Business websites
* Event promotion pages

---

## 4. Project Objectives

The main goals of this project are:

1. Learn how to structure a webpage using HTML
2. Apply CSS to design banners, sections, and layouts
3. Organize project files properly
4. Use images and icons effectively
5. Create a responsive and visually attractive webpage
